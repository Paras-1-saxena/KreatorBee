Website Instamojo Payment Acquirer

The module adds Instamojo Payment method for customers on the Odoo website. It offers seamless integration Of Instamojo with Odoo so you can accept online payments from customers using instamojo on the website. To make tracking easier, transaction details of each payment is stored in Odoo.
